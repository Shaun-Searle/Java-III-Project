Usage


UI has a standard set of media player controls with play/pause, next/previous and skip.


Load Playlist from MP3

The load playlist button allows the user to select mp3 files from the local computer. These will be loaded into the playlist with the md5 hash of the file appended to the name as a unique identifier

Double clicking a track will play the selected directly otherwise the controls are used to navigate

Tools Tab

Search

Inputting the file name and extension (Excluding the hash) will highlight the song in the list if present

Save/Load as Json

Selecting Save to JSON will bring up a file dialog to pick a location. This will save the current playlist to a json file that can be loaded in the same way using Load button.
